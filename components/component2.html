Footers

The footer is included on all pages within this design system so users can easily view copyright information and navigate to social media links related to my design work. It will also be responsive and the below images display the Desktop, Tablet, and Mobile views of the footer.

Accessibility

Similar to the navigation bar, this design system also has a region at the bottom of every page that contains site-wide information, including copyright information and social media links. The <footer> element should be used to define this region.


Usage

A footer must be included in all pages created within this design system.

Images

The first set of images included below are headshots in desktop, tablet, and mobile responsive sizes for all screen views. Below this, icons are shown including from start to finish an email icon, LinkedIn icon, GitHub icon, and a hamburger menu.


Headshots


Icons


Accessibility

All images should include an <img src> tag including alt text to show the description of the image. All images should also include a link to an external or internal webpage using <a href>.


Usage

The headshot images are used for displaying on the About page in order to put a face to name. The social media icon images are used for linking social media pages. The hamburger menu is used as a navigation bar for mobile screens.

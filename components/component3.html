Cards

Cards are used within this design system to link to examples of work. Cards include an image, project title, and summary describing the content, as well as a link to the content from the Project Title.

Accessibility

The cards are interactive and follow a link to each project page. The cards sit in a list in order to display multiple design projects. This is important as screen-readers enumerate items in lists.


Usage

The cards will be used under the "Work" page where all design case studies will be displayed and linked to individual project pages.

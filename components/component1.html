Navigation Bars

The navigation bar is included on all pages within this design system so users can easily navigate to different pages including the About, Work, and Contact pages. It will also be responsive and the below images display the Desktop, Tablet, and Mobile views of the navigation bar.

Accessibility

In order to ensure accessibility, using link tags for navigation is necessary. Links need to be within an unordered list structure so that a screen reader will read out how many things are in the list to give visually impaired users the most information possible about the contents of the navigation. Use the semantic <nav> tag around the unordered list and links with an ARIA role='navigation' attribute. The color palette should be utilized properly to ensure contrast is high enough for readability.


Usage

A navigation bar must be included in all pages created within this design system.


        
Code Example

        <nav class="menu">

             <img src="../img/Logo.png" alt="Logo">

                 <ul>
                    <li><a href="index.html">About</a></li>

                    <li><a href="work.html">Work</a></li>

                    <li><a href="contact.html">Contact</a></li>

                </ul>
            </nav>
